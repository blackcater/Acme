# @acme/code-agent

`@acme/code-agent` is the Acme-native code agent implementation. It extends
the base agent runtime with deep integration for subagents, skills, commands,
and MCP servers.

**Key Features:**
- Subagent orchestration and management
- Skill system for extensible capabilities
- Command execution with security validation
- MCP (Model Context Protocol) client integration
- File system operations and git integration
- Terminal and LSP integration

**Use Cases:** Building custom code agents, extending Acme capabilities
