# @acme/shared

`@acme/shared` provides cross-platform utility functions and shared code used
across all Acme applications. It includes common helpers for filesystem
operations, date formatting, validation, and more.

**Key Exports:**
- File system utilities
- Date/time formatting
- String and encoding helpers
- Validation utilities
- Common data structures

**Note:** This package may be split into focused utility packages in the future.
